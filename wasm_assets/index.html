Goal

Send your boats to collect as many fish as possible. You have 300 turns for that.

The boats are controlled by executing commands via a update_player (i, world) function. "up", "down", "left" and "right" are valid return values. In case of no or invalid output the boat will wait for a turn.

World reference

World table

  • fish - 1-indexed list of Fish objects
  • home - home tile position {x, y}
  • players - 1-indexed list of Player objects
  • obstacles - 1-indexed list of obstacle positions

Fish table

  • x, y - current position
  • life - turns of life left
  • value - fish score / value

Player table

  • x, y - current position
  • loaded - collected Fish object or nil

Custom functions

  • debug(value) - log any Lua value to a side console
