FishBots

A Lua coding game

Info

A tiny coding game, where your goal is to catch as many fish as possible in 300 turns.

World reference

World table

  • fish - 1-indexed list of Fish objects
  • home - home tile position {x, y}
  • fish - 1-indexed list of Player objects

Fish table

  • x, y - current position
  • life - turns of life left
  • value - fish score (currently always 1)

Player table

  • x, y - current position
  • loaded - collected Fish object or nil

Custom functions

  • debug(value) - log any Lua value to a side console
